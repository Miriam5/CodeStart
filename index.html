There are five basic elements in computing. They'll come up in all the tutorials and it can help to know about them before you start.

Loops let you repeat pieces of programs forever, until something changes or for a certain number of times. Conditionals are if-then statements. Functions are like the engine. They're how you tell the program what to do. Each computing language comes with functions, and you can define your own functions, too.

Loops, conditionals and functions. That's three. Then there are variables. This one can be tough for beginners -- variables are like storage spaces for a piece of data. Imagine a bus. It picks up a passenger, and drops her off somewhere. Then the passenger gets in a train. And the bus picks up someone else. Variables hold something till you need it, or want to change it. Variables hold one item at a time. Lists, also called arrays, hold more than one thing. (And if you have lots of info, you'll probably need a data base; that comes later). Oh, and libraries. Just like book libraries, they're collections of functions you can call up, so you don't have to write your own.

Learn the Basics

Show them to TechGirlz which has lessons that teachers, librarians and other adults in your neighborhood can teach to you and your friends. Need more? Try Code.org's teacher pages or Google's CS-First Club.

Scratch.org

Scratch uses colorful blocks to build code.Start with the step-by-step tutorials (look for the ? along the right side). Take your time. There's so much you can do and learn on Scratch. (Works on laptops and desktops, but not on phones and tablets.)

Hour of Code

Hour of Code has many short lessons, in different languages. Start anywhere. The goal? Get comfortable. Use block-coding, or click to see the text-code underneath. Feel how a program thinks and builds with code. How many can you do?

Code Combat

Code Combat is a fantasy game where you move through by coding answers. The first levels are free, and then they ask you to pay. Ask your school to buy it for everyone.

Code Monkey

CodeMonkey uses CoffeeScript, which is a junior version of JavaScript. There's a free version and a paid version, which I recommend, if you can. Find it at playcodemonkey.com.

Hour of Python

It's not a snake, it's a coding language. These lessons from trinket.io get you started in Python, a back-end language that powers about 1/3 of the internet. Offered by trinket.io, which has other languages to explore, too. Big plus: trinket.io works on all devices, even your smartphone.

Khan Academy

Most websites are built from HTML, CSS and JavaScript. Why three languages? HTML gives structure. CSS gives style. JavaScript makes it do things. Try the lessons in JS: Drawing & Animation, and Intro to HTML/CSS: Making webpages, and Advanced JS: Games & Visualizations.

Dash

Want to learn more about making websites and web apps? Code and design webpages with these four lessons from Dash.

Google CS First

Google has many lessons, and you can pick from categories like art, music, animation, sports and storytelling. There's also a full class that you can teach yourself, or ask your teachers and school to offer.

Make an App

On AppInventor you can design and code an Android app. Definitely watch and code along with the video tutorials.

Codeacademy

Great place to learn Python, Ruby, JavaScript and how to make a website and many more. Kids: this is where adults learn. Tip: start with Python, or start with HTML/CSS and then move on to the JavaScript tutorials.

Are you ready for more? Try FreeCodeCamp or sign up for a computer science, design or coding class at a local college. Definitely find a friend to code with, and let me know how it goes.
